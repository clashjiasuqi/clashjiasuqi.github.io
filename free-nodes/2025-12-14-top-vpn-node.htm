什么是科学上网？

科学上网是指使用特定的工具或技术手段，通过绕过网络限制，访问那些在某些国家和地区被屏蔽或限制的网站。这些工具通常采用加密协议来保护用户的隐私，同时提供更快的网络连接速度。常见的科学上网工具包括VPN（虚拟私人网络）、代理服务器、Shadowsocks、V2Ray等。

在选择科学上网工具时，安全性、速度、功能和价格是最重要的考量因素。接下来，我们将为大家介绍2023年表现最佳的免费和付费科学上网工具，并对它们的优缺点进行详细分析。

FAQ

WinXray 支持 Trojan 协议吗？
WinXray 内核基于 Xray-core，因此原生支持 Trojan 协议。用户在导入配置时，只需选择 Trojan 类型节点即可，无需额外插件，非常适合需要 TLS 伪装的环境。
PassWall2 的延迟测试如何影响策略组选择？
PassWall2 会根据节点延迟结果更新策略组的优先节点，自动选择延迟最低的节点处理流量。这样可以优化访问速度，提高网页浏览、视频和游戏体验的稳定性。
Trojan 的伪装域名配置有哪些注意事项？
伪装域名必须真实可访问且证书有效，否则无法建立连接。正确配置可隐藏真实流量特征，降低被封锁和检测风险，提高节点安全性和长期可用性。
Mellow 是否支持自定义 DNS？
支持。Mellow 的配置文件中可设置 DNS 服务器地址及模式（如 DoH、DoT）。用户可使用安全 DNS，如 Cloudflare 1.1.1.1 或 Google DNS，以减少污染风险。
VMess 的 WebSocket + TLS 配置有哪些好处？
WebSocket + TLS 可伪装为普通 HTTPS 流量，增强抗封锁能力和隐蔽性。结合域名伪装和策略组使用，可以在高封锁网络中保持稳定、高速和安全的网络访问体验。
Sing-box 支持按 IP 段分流吗？
支持。Sing-box 可根据 IP 段将国内流量直连，国外流量走代理，结合策略组实现智能分流。这样可保证访问速度和稳定性，同时优化节点使用效率，提高网络体验。
为什么某些网站在通过代理后登录会频繁要求二次验证？
代理会改变访问 IP 与地理位置，目标站点可能检测到异常登录并触发验证码或二次验证。建议对常用账户使用白名单 IP 或在代理配置中对特定站点设置直连来避免频繁验证。
VLESS 的 Flow 参数有什么作用？
Flow 参数用于控制 VLESS 的传输模式，如 XTLS Vision 或 Reality 模式。它能影响性能、安全性与兼容性，需根据客户端支持情况正确匹配配置。
Netch 的全局代理与规则代理有何不同？
全局代理将所有流量都通过节点转发，而规则代理只针对指定 IP、端口或应用。全局代理适合游戏和全局科学上网，规则代理适合优化带宽和分流特定流量。
SSR 节点和 V2Ray 节点可以混合使用吗？
可以，但要使用支持多协议的客户端，如 Clash、Sing-box、Mihomo 等。这类客户端能在同一配置文件中加载不同协议的节点，通过规则分流让 SSR 与 V2Ray 同时工作。
